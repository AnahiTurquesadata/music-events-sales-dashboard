# 🎵 Music Events Sales Dashboard

## Project Overview

This project analyzes the revenue performance of 10 outdoor electronic music events organized by an Argentine event production company during 2024–2025, using SQL, BigQuery, and Looker Studio. The analysis explores sales trends, ticket performance, customer behavior, and key revenue insights to support data-driven decision-making.


## Tools

- SQL
- Google BigQuery
- Looker Studio
- Excel Sheet

## Dashboard

- Sales Performance
- Customer Analysis
